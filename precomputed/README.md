# Precomputed Evaluation Tables

These CSVs are **intentionally tracked** in the repository. They are the default
reproducibility layer for the figure and table notebooks (01–04): a reader can
clone the repository, install dependencies, and reproduce every manuscript
figure and table in a few minutes without running CPA or GEARS training.

## Contents

| Subdirectory | Contents |
|---|---|
| `eval/` | Per-seed evaluation tables from the full n=100 CPA sweep and the matched-n GEARS audit |
| `tables/` | Table 1 (mechanism summary) and Table 2 (GEARS matched-n verdict) as CSVs |
| `figure_inputs/` | Aggregated inputs for Figure 1 panels (per-seed severity Pearson, winsorisation summary, LOO sensitivity) |

## Provenance

These CSVs were generated by the analysis pipeline (`P01–P04` notebooks) on
the final pre-submission run. Re-running the pipeline with `--analysis` or
`--train-full` writes the corresponding files to `data/`; the shipped versions
in `precomputed/` are left unchanged so the figure notebooks can always fall
back to them.

## Why these are tracked

The full retraining pipeline takes approximately 13 hours on a single Apple
Silicon CPU. Shipping the lightweight (~MB) evaluation CSVs lets the
reproducibility flow be:

```
clone  →  uv sync  →  ./run_all.sh --figures
```

rather than:

```
clone  →  uv sync  →  13 hours of training  →  ./run_all.sh --figures
```

The heavy per-seed prediction artifacts (h5ad files) are intentionally not
shipped here. They are regenerated by `P01_train_cpa.ipynb` and
`P02_train_gears.ipynb` when running in `--train-full` mode and are written
to `data/predictions/` (gitignored).

## Regeneration

To regenerate these tables from existing per-seed predictions:

```bash
./run_all.sh --analysis
cp data/eval/*.csv precomputed/eval/
cp data/tables/*.csv precomputed/tables/
cp data/figure_inputs/*.csv precomputed/figure_inputs/
```

To regenerate from scratch (requires CPA + GEARS dependency groups and ~13 hours):

```bash
uv sync --group cpa --group gears
./run_all.sh --train-full
cp data/eval/*.csv precomputed/eval/
cp data/tables/*.csv precomputed/tables/
cp data/figure_inputs/*.csv precomputed/figure_inputs/
```
